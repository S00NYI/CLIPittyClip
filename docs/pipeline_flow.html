💡 Pipeline Notes

  • Standard Path (4a): Uses bedtools bamtobed - fast but no mutation tracking
  • CIMS/CITS Path (4b): Uses parseAlignment.pl - captures mutations for crosslink analysis
  • CIMS: Detects mutation sites where RT introduced errors at crosslinks
  • CITS: Detects truncation sites where RT stopped at crosslinks
